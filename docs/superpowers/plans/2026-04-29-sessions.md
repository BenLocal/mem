# Sessions Auto-Bucket Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `sessions` table and per-`(tenant, caller_agent)` auto-bucket on ingest. Each new memory carries a `session_id`; the resolver continues an active session if its `last_seen_at` is < `MEM_SESSION_IDLE_MINUTES` ago, otherwise closes it and opens a new one.

**Architecture:** New `db/schema/004_sessions.sql` migration adds the `sessions` table and `memories.session_id` column. New `domain/session.rs` carries the `Session` type. Repository trait gains four methods (`latest_active_session`, `open_session`, `close_session`, `touch_session`) implemented against DuckDB. New `pipeline/session.rs` exposes `resolve_session` (DB-bound) plus a pure `decide_session` (unit-testable). `memory_service::ingest` calls `resolve_session` before record construction and `touch_session` after a successful write.

**Tech Stack:** Rust 2021, DuckDB (bundled), `uuid::Uuid::now_v7()` (already a dep), the existing `Repository` trait pattern.

**Spec:** `docs/superpowers/specs/2026-04-29-sessions-design.md`

```rust
async fn latest_active_session(
    &self,
    tenant: &str,
    caller_agent: &str,
) -> Result<Option<Session>, StorageError> {
    let conn = self.connection.lock().await;
    let mut stmt = conn.prepare(
        "SELECT session_id, tenant, caller_agent, started_at, last_seen_at,
                ended_at, goal, memory_count
         FROM sessions
         WHERE tenant = ? AND caller_agent = ? AND ended_at IS NULL
         ORDER BY last_seen_at DESC
         LIMIT 1",
    ).map_err(StorageError::from)?;
    let mut rows = stmt.query([tenant, caller_agent]).map_err(StorageError::from)?;
    if let Some(row) = rows.next().map_err(StorageError::from)? {
        Ok(Some(Session {
            session_id: row.get(0)?,
            tenant: row.get(1)?,
            caller_agent: row.get(2)?,
            started_at: row.get(3)?,
            last_seen_at: row.get(4)?,
            ended_at: row.get(5)?,
            goal: row.get(6)?,
            memory_count: row.get::<_, i64>(7)? as u32,
        }))
    } else {
        Ok(None)
    }
}

async fn open_session(
    &self,
    session_id: &str,
    tenant: &str,
    caller_agent: &str,
    now: &str,
) -> Result<Session, StorageError> {
    let conn = self.connection.lock().await;
    conn.execute(
        "INSERT INTO sessions (session_id, tenant, caller_agent, started_at, last_seen_at, memory_count)
         VALUES (?, ?, ?, ?, ?, 0)",
        [session_id, tenant, caller_agent, now, now],
    ).map_err(StorageError::from)?;

    Ok(Session {
        session_id: session_id.to_string(),
        tenant: tenant.to_string(),
        caller_agent: caller_agent.to_string(),
        started_at: now.to_string(),
        last_seen_at: now.to_string(),
        ended_at: None,
        goal: None,
        memory_count: 0,
    })
}

async fn close_session(
    &self,
    session_id: &str,
    ended_at: &str,
) -> Result<(), StorageError> {
    let conn = self.connection.lock().await;
    conn.execute(
        "UPDATE sessions SET ended_at = ? WHERE session_id = ? AND ended_at IS NULL",
        [ended_at, session_id],
    ).map_err(StorageError::from)?;
    Ok(())
}

async fn touch_session(
    &self,
    session_id: &str,
    last_seen_at: &str,
) -> Result<(), StorageError> {
    let conn = self.connection.lock().await;
    conn.execute(
        "UPDATE sessions
         SET last_seen_at = ?, memory_count = memory_count + 1
         WHERE session_id = ?",
        [last_seen_at, session_id],
    ).map_err(StorageError::from)?;
    Ok(())
}
```

pub enum SessionDecision {
    Continue(String),
    OpenNew { previous: Option<String> },
}

pub fn decide_session(
    latest: Option<(&str, &str)>,
    now: &str,
    idle_minutes: u64,
) -> SessionDecision {
    match latest {
        Some((id, last_seen)) => {
            let elapsed = minutes_since(last_seen, now).unwrap_or(i64::MAX);
            if elapsed >= 0 && (elapsed as u64) < idle_minutes {
                SessionDecision::Continue(id.to_string())
            } else {
                SessionDecision::OpenNew { previous: Some(id.to_string()) }
            }
        }
        None => SessionDecision::OpenNew { previous: None },
    }
}

pub async fn resolve_session<R: Repository + ?Sized>(
    repo: &R,
    tenant: &str,
    caller_agent: &str,
    now: &str,
    idle_minutes: u64,
) -> Result<String, StorageError> {
    let latest = repo.latest_active_session(tenant, caller_agent).await?;

    let decision = decide_session(
        latest.as_ref().map(|s| (s.session_id.as_str(), s.last_seen_at.as_str())),
        now,
        idle_minutes,
    );

    match decision {
        SessionDecision::Continue(id) => Ok(id),
        SessionDecision::OpenNew { previous } => {
            if let Some(prev) = previous {
                repo.close_session(&prev, now).await?;
            }
            let new_id = uuid::Uuid::now_v7().to_string();
            repo.open_session(&new_id, tenant, caller_agent, now).await?;
            Ok(new_id)
        }
    }
}
